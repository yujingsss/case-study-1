Memory Palace, Future

A "game" not like a game

An interactive virtual environment telling a story about memories in an abandoned place from the future. With a cinematic narrative, the audience enters a futuristic space of consciousness with no human existence. The audience then dives underwater into the memory palace, picking items and view the memories from the past. It is a "game" that doesn't feel like a game.
